# Clothing Inventory Management System

Sistem manajemen inventori untuk produk pakaian dengan fitur lengkap.

## Features

- ✅ Product Management (CRUD produk dengan HPP & harga jual)
- ✅ Order Processing (pesanan customer dengan auto-calculate profit)
- ✅ Stock Management (stock in, stock out, adjustment)
- ✅ Low Stock Alerts
- ✅ Sales & Profit Reports
- ✅ User Authentication & Authorization
- ✅ Responsive Dashboard

## Tech Stack

- **Backend**: NestJS + TypeORM + PostgreSQL
- **Frontend**: React + Vite + Tailwind CSS
- **Deployment**: Docker + Docker Compose + Caddy (SSL)

## Quick Start

git clone <your-repo-url>
